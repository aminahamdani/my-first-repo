## ğŸ“š Usage

This repository contains practice files and examples for learning Git. Here's what you can do:

1. **Explore the contents** - Browse the files and understand the repository structure
2. **Experiment with Git commands** - Try different commands like `git status`, `git log`, `git branch`
3. **Create branches** - Practice creating and switching between branches
4. **Make commits** - Practice writing good commit messages
5. **Merge branches** - Learn how to merge changes safely

**Common commands to try:**
```bash
git status              # Check the status of your repository
git log                 # View commit history
git branch              # List all branches
git checkout -b <name>  # Create a new branch
git add .               # Stage changes
git commit -m "message" # Commit changes
```
