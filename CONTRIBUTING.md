# Contributing to my-first-repo

Welcome! This is a learning repository, and contributions—especially practice pull requests—are encouraged.

## How to Contribute

### 1. Fork & Clone
```bash
git clone https://github.com/yourusername/my-first-repo.git
cd my-first-repo
```

### 2. Create a Branch
Use a descriptive branch name:
```bash
git checkout -b feature/add-example
# or
git checkout -b fix/typo-in-readme
```

### 3. Make Changes
- Add example scripts, fix typos, improve docs, or expand practice exercises.
- Keep changes small and focused.

### 4. Commit
Write clear commit messages:
```bash
git commit -m "Add: simple Python counter example"
git commit -m "Fix: typo in Installation section"
```

Good commit message format:
- **Add:** New feature or file
- **Fix:** Bug or error correction
- **Update:** Improvements or clarifications
- **Docs:** Documentation changes

### 5. Push & Open a Pull Request
```bash
git push origin feature/add-example
```
Open a PR on GitHub with a brief description of your change.

## What to Contribute

- Example scripts or practice exercises
- Typo fixes or grammar improvements
- Clarifications to the README or docs
- New learning resources or links
- Answers to common Git/GitHub questions

## Code of Conduct

Please review [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md) before contributing.

## Questions?

Open an Issue or reach out—this is a learning space, so all questions are welcome!
